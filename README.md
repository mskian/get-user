# Get User

API for Name Wishing Site

> Supports Multi Language

## Usage

- Download or Clone the repo

```sh
git clone https://github.com/mskian/get-user.git
cd get-user
yarn
```

- Start and test

```sh
yarn start
```

```html
http://localhost:4000/santhosh%20veer
```

```json
[{"content":"santhosh veer"}]
```
